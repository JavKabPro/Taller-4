\# Recetario Digital



Desarrollo de una App web de búsqueda de recetas con diseño moderno y tarjetas visuales atractivas. Permite buscar recetas y visualizar resultados en tarjetas con imágenes de comida.



\##  Descripción



\*\*Recetario Digital\*\* es una aplicación (frontend) que simula una app de búsqueda de recetas. 

Cuenta con:



\- Barra de búsqueda en la parte superior

\- Tarjetas visuales con fotos de comida

\- Diseño limpio y moderno

\- Diseño responsive



Actualmente funciona con datos simulados (sujeta a revisión), pero está pensada para ser optimizada.



\## Funcionalidades Actuales



\- Buscador de recetas

\- Calidad de imágenes y tarjetas dinámicas

\- Interfaz visual atractiva

\- Diseño adaptable (Responsive)



\##  Funcionalidades Futuras



\- Consumir una API real de recetas

\- Guardar recetas favoritas

\- Integración con Firebase

\- Autenticación de usuarios

\- Filtro por categorías





\## Tecnologías Utilizadas



\- HTML5

\- CSS3

\- JavaScript

\- (Opcional: React)

\- (Opcional: Firebase en el futuro)
